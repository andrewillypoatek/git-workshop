Git Cheatsheet

Git/GitHub Workshop — keep this open during the exercises

Exercise 1 — Fork & Clone

git clone <url> hello-world

Download a repository from GitHub to your machine. Use the SSH URL (starts with git@github.com:), not HTTPS. The hello-world at the end names the local folder.

cd <folder-name>

Navigate into the cloned folder.

git status

See the current state of your repo. Run this whenever you're unsure.

Exercise 2 — Edit, Commit, Push

git add .

Stage all changed files, ready to be committed.

git add <filename>

Stage a specific file only.

git commit -m "your message"

Save a snapshot with a description of what changed.

git push

Upload your commits to GitHub.

Exercise 3 — Branch & PR

git checkout -b <branch-name>

Create a new branch and switch to it. Use feature/your-name as the name.

git branch

List all branches. The active one is marked with *.

git push -u origin <branch-name>

Push a new branch to GitHub for the first time.

Good to know

git log --oneline

See the commit history in a compact view.

git diff

See exactly what changed since your last commit.

git pull

Download and apply the latest changes from GitHub.

Tip: Run git status constantly — before and after every step. It's the best way to understand where you are.
