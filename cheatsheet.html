Git commands,
all in one place

Every command you'll touch during the exercises. Bookmark it and keep it open in a tab.

the commands
Ex 1 — Fork & Clone
git clone <url> hello-world

Download a repo from GitHub. Use the SSH URL (starts with git@github.com:), not HTTPS. The last word names the local folder.

cd <folder-name>

Navigate into the cloned folder.

git status

See the current state of your repo. Run it whenever you're unsure.

Ex 2 — Edit, Commit, Push
git add .

Stage all changed files, ready to commit.

git add <filename>

Stage a specific file only.

git commit -m "message"

Save a snapshot with a description of what changed.

git push

Upload your commits to GitHub.

Ex 3 — Branch & PR
git checkout -b <name>

Create a branch and switch to it. Use feature/your-name.

git branch

List all branches. The active one is marked with *.

git push -u origin <name>

Push a new branch to GitHub for the first time.

Good to know
git log --oneline

See the commit history in a compact view.

git diff

See exactly what changed since your last commit.

git pull

Download and apply the latest changes from GitHub.

Tip — Run git status constantly, before and after every step. It's the fastest way to understand where you are.
