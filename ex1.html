Fork & Clone

0:05 → 0:15

Get the instructor's code onto your machine. You'll fork it first (making your own copy on GitHub), then clone it locally.

Repo to fork
andrewillypoatek / learning-github
Open on GitHub →
1

Fork the repo

On the repo page, click Fork (top right). This creates your own copy under your GitHub account.

Important: Clone your fork, not the instructor's repo. Your fork URL will have your username in it.
2

Copy the SSH URL

On your fork, click the green Code button → select SSH → copy the URL. It should look like:

git@github.com:YOUR-USERNAME/learning-github.git
3

Clone it locally

In your terminal, navigate to where you want the folder, then run:

git clone git@github.com:YOUR-USERNAME/learning-github.git hello-world

The hello-world at the end names the local folder.

4

Enter the folder

cd hello-world
5

Open the page in your browser

Find index.html in Finder and double-click it, or drag it into a browser window. You should see the Hello World page.

Tip: Run git status — it should say "nothing to commit, working tree clean".
