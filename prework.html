Set up before
the workshop

Three quick things before we meet. We'll handle Git and SSH together at the start of the session.

what you need
1

Create a GitHub account

Go to github.com and sign up. If you already have one, you're good.

Tip — Use a personal email you'll keep long-term. Your GitHub username is visible publicly.
2

Install Visual Studio Code

Download and install it from code.visualstudio.com. It's free.

Already have an editor? No problem — bring what you use. VS Code is just the recommendation for anyone starting fresh.
3

Make sure you can log in to GitHub

On the day of the workshop, have github.com open and be logged in. We'll need it during the setup block.

✓ that's it — we'll do the rest together
