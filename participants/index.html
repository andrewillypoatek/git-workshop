Add yourself to the Team

By the end of this you'll have opened your first Pull Request and your page will appear in the gallery below.

what you'll do
1
Clone the repo

Open your terminal in the folder where you want to save the project (Desktop is fine). Then download the workshop repo:

git clone git@github.com:andrewillypoatek/git-workshop.git

Then enter the folder:

cd git-workshop
2
Create a branch

Never work directly on main. Create your own branch first:

git checkout -b add/your-name
3
Create your page

Duplicate the template and rename it with your own name:

cp participants/template.html participants/your-name.html
4
Edit your page

Open participants/your-name.html in your editor and fill in the sections marked with ✏️:

  • 🖼️ Hero image — swap the gif URL for your own photo or a gif that represents you
  • 👤 Name — your real name
  • 💼 Role — your job title
  • 📝 About — one or two sentences about yourself
  • ⚡ Fun fact — something people might not know about you
  • 🎨 Accent color — change --accent at the top to any color you like
5
Commit and push your branch

Stage your new file, commit it, and push your branch to GitHub:

git add participants/your-name.html
git commit -m "Add my participant page"
git push -u origin add/your-name
6
Open a Pull Request

Go to github.com/andrewillypoatek/git-workshop. You'll see a prompt to open a Pull Request from your branch — click it, add a short description, and submit. Once it's merged, your page will show up in the gallery!

💡 Stuck? Check the Git cheatsheet or ask for help.
