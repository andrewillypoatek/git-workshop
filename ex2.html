ex.02

Edit · Commit
· Push

0:15 → 0:32

Make changes to the Hello World page, save a snapshot with Git, and upload it to GitHub. This is the core loop you'll use every day.

part 1 — follow along
1

Fill in your details in index.html

Open index.html in a text editor. Look for the ✏️ comments and replace the placeholder text with your own name, role, about, and fun fact. Swap the hero image URL for your own too.

2

Change the background color in style.css

Open style.css. Near the top, find the --background-color variable and change its value.

Tip — Try #1a1a2e, #f0f4f8, or any hex you like.
3

Stage your changes

git add .

This marks all changed files as ready to commit.

4

Commit with a message

git commit -m "Add my name and bg color"

The message is a note to your future self. Make it descriptive.

5

Push to GitHub

git push
Verify — Go to your fork on GitHub; you should see the commit message next to the changed files.
